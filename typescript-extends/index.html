TypeScript实现继承的方式

March 16, 2018

先来明确几个问题

Q1: 什么是继承
A1: 继承（英语：inheritance）是面向对象软件技术当中的一个概念。如果一个类别A“继承自”另一个类别B，就把这个A称为“B的子类别”，而把B称为“A的父类别”也可以称“B是A的超类”。继承可以使得子类别具有父类别的各种属性和方法，而不需要再次编写相同的代码。在令子类别继承父类别的同时，可以重新定义某些属性，并重写某些方法，即覆盖父类别的原有属性和方法，使其获得与父类别不同的功能。另外，为子类别追加新的属性和方法也是常见的做法。form wiki

Q2: TypeScript是怎么来实现的类继承呢?
A2: 寄生组合继承

分析

  • 子类要有父类的属性方法(静态，实例，原型链)，还可以重写父类的属性和方法
  • 寄生组合继承

TypeScript的实现

源文件 extend.ts

class Animal {
  static type = "An";
  constructor(protected name = "Animal") {}
}
class Cat extends Animal {
  constructor(name?: string) {
    super(name);
  }
  sleep() {
    console.log("sleep");
  }
}
